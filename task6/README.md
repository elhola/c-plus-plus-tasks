Функцiї
1)	Створити функцію(метод), яка буде приймати ціле число і повертати його половину.
2)	Створити функцію, яка буде виводити в консоль повідомлення “Hello, c++!”.
3)	Створити функцію, яка буде приймати ім’я користувача та виводити привітання в консоль.
4)	Створити функцію, яка буде приймати два числа і повертати максимальне.
5)	Створити функції для введення та виведення масиву з консолі. Також створити функцію, яка буде виводити масив в оберненому порядку.
6)	Створити три функції, які будуть сортувати масив. Використати алгоритми сортування бульбашкою(wiki) , злиттям(wiki) та швидке сортування(wiki).
7)	Створити функцію, яка буде повертати мінімальний елемент масиву.
8)	Створити дві функції з одним іменем. Перша буде приймати ціле число і повертати true якщо число парне. Друга функція буде приймати string і повертати true, якщо довжина цього рядка це парне число.
